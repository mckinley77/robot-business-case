# robot-business-caseRobotic Mowing Business Case Generator
Version 0.9 Beta

An independent decision-support tool for evaluating the financial and operational benefits of introducing hybrid robotic mowing.

Purpose

The Generator has been developed primarily for golf clubs but may also be applicable to other professionally managed turf areas.

It helps organisations evaluate the potential benefits of hybrid robotic mowing, including:

reduced routine mowing costs
reduced environmental impact
reduced winter damage
release of skilled staff capacity
improved long-term course presentation
Status

This is a Beta release intended for evaluation by a small number of users.

Feedback is welcomed and will be used to improve future versions.
